Authentication

If you do not have an account as it is, get one using one of the partners below :

Accounts by default can only access certain endpoints. Accounts can be given permissions to other endpoints by an administrator.

Locked endpoints and upgraded accounts can be acquired through Patreon subscriptions. Check out the plans Here For any support with your account, or any part of the api, join the Discord server

Account Flags

Flag Purpose
ADMIN Access to all endpoints (including some undocumented ones)
UPGRADED Higher limits and some special endpoints
EXECUTE Access to the execute endpoint
IP Access to the IP endpoint
WOWDB Access to the WoWDB endpoints

Invalidate Token

URL GET /auth/invalidate

Invalidates your current token and gives you a new one.

Information Request

URL GET /auth/info

Fetch information related to your token. such as the email, flags, and id of the account.
